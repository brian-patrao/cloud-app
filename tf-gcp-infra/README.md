# tf-gcp-infra

### GCP Setup
1. Install Google CLI
2. gcloud init
3. gcloud projects create PROJECT_ID --name=PROJECT_NAME

### Terraform 
1. terraform init
2. Create files
3. terraform plan
4. terraform fmt
5. terraform validate
6. terraform apply
7. Run with terraform apply -var='vpc_list=["vpc1","vpc2"]'
